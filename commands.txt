1. Command palette: cmd/ctrl+p
Przechodzenie po wszystkich plikach po nazwie, ostatnio otwarty jest na samej górze

2. Run Commands: cmd/ctrl+p   >
Dostęp do komend, rozszerzeń

3. Find symbols (class, interface, elements, variables) in the command palette: cmd/ctrl+p   @

4. The same like upper but open in the file: cmd/ctrl+shift+.

5. Global symbols in the all files in command palette: cmd/ctrl+p   #
can use short of name symbols, by camelcase

6. Search content in all files in search bar: cmd/ctrl+shift+f

7. Go to line: cmd/ctrl+p :12.

8. Zaznaczanie takich samych wyrażeń: cmd/ctrl+d

9. Multiline texting: alt+mouse

10. Przenoszenie linii: alt+up/down

11. Kopiowanie linii: shift+alt+up/down

12. Zaznaczanie linii po linii: cmd/ctrl+l

13. Comment: cmd/ctrl+'/'

14. Open terminal: cmd/ctrl+'

15. Podział ekranu: cmd/ctrl+\

16. Otwarcie terminalu: mac: shift/ctrl+`(~)

17. Czyszczenie terminala: cmd/ctrl+k
